Java Projects

Command Line Wordle (Java)

This project is a spin-off of the popular game Wordle, displayed in the command line. The user has 6 attempts to guess a 5-letter word. Correct letters in the correct position are displayed as capital letters, correct letters in the wrong position as lowercase letters, and "X" placeholders for incorrect guesses. This project helped me learn about string comparison and game logic in Java.

Measures of Central Tendency with GUI (Java)

This project takes integer input data from the user and calculates the measures of central tendency. The data is stored in an ArrayList and the mean, number of data points, standard deviation, and variance are calculated. I learned to work with ArrayLists, perform statistical calculations, and build a GUI using JavaFX and SceneBuilder.

Graphical Data Display with GUI (Java)

Building on my Measures of Central Tendency project, this project adds animation to a bar graph. The array list contains integers entered by the user, calculates measures of central tendency, and adjusts the bar graph for frequency upon submission. I learned animation techniques with JavaFX.

Two Player Air-hockey (Java)

In the process of uploading. This project is a two-player air hockey game where the winner is the first to reach 5 points. It helped me learn physics for ball movement and collision, as well as gameplay functions like scoring and determining the winner.

Single Player Air-hockey vs AI Opponent (Java)

This project is a single-player air hockey game with an AI opponent, played until the first player reaches 5 points. The ball's positioning and trajectory are stored in a hashmap. The AI's movements are based on a learning factor and the position data from the hashmap. I learned key commands with JavaFX, data storage in hashmaps, and basic AI development.
