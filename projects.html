Command Line Hangman(Python)
Revising. This project gives the user 7 errors to guess the letters in the target word. If the user guesses a correct letter then the letter will show up in the correct position in the word, otherwise, if the guess is incorrect, one of the seven lives is subtracted and nothing happens to the display to help the user figure out the solution. This was one of my first python proejcts. I learned how to compare strings, create gameplay logic, and store the responses in an array.

Measures of Central Tendency with GUI(Java)
This project takes an input of data (integers) from the user and calculates the measures of central tendencies of the data set when the calculate button is clicked. The calculator first takes the data and puts it into an arrayList, then solves for the mean, # of data points, standard deviation, and variance. This project helped me learn to create and adjust ArrayLists to store Data, code calculation function, and build a GUI using JavaFX and SceneBuilder.

This project helped me learn animation with JavaFX.
